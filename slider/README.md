# rotateSlider
A rotateSlider using CSS3 &amp; jQuery .
### 目前还未开发完成，如需使用请Copy Scss源码以及JS

###用法
1. 引入`src/js/jquery.rotateSlider.js`
2. 引入`src/css/_common.scss` (也可以是编译好的`assets/css/app.min.css`)
3. 在你的JS中调用
```
  $('#slider').rotateSlider();
  // 你可以传入一个对象作为参数，TODO:增加更多自定义参数
```
